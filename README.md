## インストール

[こちら](https://chromewebstore.google.com/detail/kcojgobbcanjkbhgmhecdfoaakgedcjd)からインストールできます。

## 機能

「Show More Locations」のボタンが画面内に表示されると自動でボタンを押し、フレンドの居るインスタンスを読み込みます。
